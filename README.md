# Progetto TecWeb

Per creare l'ambiente
```bash
docker compose up --build
```

Per distruggere l'ambiente
```bash
docker compose down
```

Per distruggere l'ambiente e rimuovere i volumi associati (dati persistenti del database)
```bash
docker compose down --volumes
```
